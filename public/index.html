✦ How to set it up on iPhone (iOS Shortcuts)
1
Copy your URL above

Click "Generate My Wallpaper URL" and then "Copy URL". This URL is personal to your city and phone — it fetches today's Suhoor & Iftar times every time it's called.

2
Open the Shortcuts app → Automation tab

Tap the + button in the top right. Select "New Automation". Choose "Time of Day". To see the sky transition from night to day, set the time to 4:00 AM (before Suhoor). Set it to repeat Daily. Most importantly: select "Run Immediately" so it runs silently in the background.

3
Add two actions to the automation

Tap "Create New Shortcut". Search for and add:

Action 1: "Get Contents of URL" — paste your copied URL here.

Action 2: "Set Wallpaper Photo" — set it to use "Contents of URL" from the previous action. Tap the chevron (›) and turn OFF "Show Preview" and "Crop to Subject". Set Wallpaper Location to Lock Screen only.

Action 1: Get Contents of URL URL: [paste your URL here] Action 2: Set Wallpaper Photo Photo: Contents of URL (from Action 1) Location: Lock Screen ✓ | Home Screen ✗ Show Preview: OFF Crop to Subject: OFF
4
Run it once manually to test

Tap the ▶ Play button in the Shortcuts editor to run it immediately. Your lock screen will update within a few seconds to display today's prayer times and the current sky color!

5
Duplicate to see the sky change! ✨

Our wallpaper naturally changes colors based on the time of day (morning blue, sunset orange, etc). To see this happen automatically, go back to your Automations list, swipe left on your 4:00 AM automation, and select Duplicate. Edit the new automation's time to your approximate Maghrib time (e.g. 5:30 PM). Now you'll get a fresh orange sunset background right at Iftar!

📱 Android users

Use Tasker (paid, ~$3) or MacroDroid (free) to create a daily trigger at 4 AM that: downloads the wallpaper URL → saves to a file → applies as wallpaper via the "Set Wallpaper" action. In Tasker: Trigger: Time 04:00 → Action: HTTP Get [your URL] → Save to file → Set Wallpaper. All three apps support this workflow natively.
